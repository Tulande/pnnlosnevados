Ficha Técnica y Proceso Metodológico

Este documento detalla la metodología empleada para el desarrollo de la plataforma interactiva de análisis de las transformaciones del paisaje en el PNN Los Nevados, presentada para la categoría académica del Premio MapBiomas Colombia 2025.

Resumen y Pregunta de Investigación

Pregunta de Investigación: ¿Cómo han evolucionado las coberturas del suelo en el PNN Los Nevados entre 1985 y 2023, y cómo se pueden comunicar eficazmente estas dinámicas a través de una herramienta digital interactiva?

Resumen: Este proyecto presenta un aplicativo web (dashboard) que visualiza y cuantifica las transformaciones del paisaje en el Parque Nacional Natural Los Nevados utilizando la serie temporal completa (1985-2023) de la Colección 2 de MapBiomas Colombia. El objetivo principal es democratizar el acceso a esta valiosa información geoespacial, ofreciendo una herramienta intuitiva que permite a usuarios no especializados explorar los cambios en coberturas críticas como glaciares y páramos. La plataforma facilita el análisis de la distribución de coberturas por año y la comparación de la evolución temporal entre dos fechas seleccionadas, traduciendo datos complejos en estadísticas y mapas interactivos comprensibles. Este trabajo constituye una aplicación directa de los datos de MapBiomas para la educación ambiental, el soporte a la toma de decisiones y la divulgación científica.

Objetivos del Trabajo

  • General: Desarrollar una plataforma web interactiva para visualizar y analizar los cambios de cobertura en el PNN Los Nevados entre 1985 y 2023, utilizando los datos de MapBiomas Colombia.
  • Específicos:
    • - Procesar y analizar los datos de MapBiomas para cuantificar las áreas de cada cobertura en diferentes años.
    • - Diseñar una interfaz web intuitiva que permita la exploración anual y la comparación temporal de mapas y estadísticas.
    • - Comunicar de forma clara y visual la magnitud de la pérdida de coberturas críticas y contextualizar los cambios observados con base en literatura científica.

Metodología

El flujo de trabajo se dividió en tres fases principales:

  1. Procesamiento de Datos Geoespaciales: La materia prima fueron los datos ráster anuales de MapBiomas Colombia. Utilizando QGIS, se realizó un pre-procesamiento para recortar los datos al límite geográfico del PNN Los Nevados. Posteriormente, para cada año de la serie, se ejecutó un análisis cuantitativo con la herramienta "Group Stats" para calcular el área (en hectáreas) de cada clase de cobertura.
  2. Desarrollo del Aplicativo Web (Frontend): Se diseñó y construyó la plataforma utilizando tecnologías web estándar (HTML5, CSS3, JavaScript) para asegurar su accesibilidad y rendimiento. La interfaz se dividió en dos módulos principales: "Explorador Anual" y "Análisis Comparativo".
  3. Integración y Visualización: Los datos cuantitativos de la Fase 1 se integraron en JavaScript para alimentar dinámicamente las estadísticas. Cada mapa anual fue exportado desde QGIS como un mapa web interactivo autónomo (usando `qgis2web`) y cargado en la plataforma mediante.

Argumentación del Proyecto

¿De qué manera es su trabajo innovador?

La innovación de este proyecto no reside en la creación de datos primarios, sino en su traducción y democratización efectiva. Transforma la compleja y masiva colección de datos de MapBiomas en una herramienta de "inteligencia territorial" intuitiva y de libre acceso. Mientras que los datos geoespaciales suelen requerir software especializado y conocimiento técnico, esta plataforma elimina esa barrera, permitiendo que cualquier persona con una conexión a internet pueda realizar análisis exploratorios. La innovación radica en tres pilares:

  • Síntesis Visual-Analítica: Combina de forma nativa la visualización de mapas interactivos con gráficos y estadísticas cuantitativas que se actualizan dinámicamente, creando una experiencia de usuario unificada.
  • Accesibilidad Radical: Al estar construido sobre tecnologías web básicas (HTML, CSS, JS), el aplicativo es extremadamente ligero, rápido y fácilmente replicable.
  • Narrativa Histórica Completa: Es una de las primeras herramientas interactivas enfocadas en un área protegida de Colombia que utiliza la totalidad de la serie temporal de 38 años.

¿Cuál es el potencial impacto de su trabajo?

El impacto de esta plataforma es multifacético y se dirige a varios sectores clave:

  • Educativo y Científico: Proporciona una herramienta didáctica lista para usar en aulas para explorar un caso de estudio real sobre cambio climático y uso del suelo.
  • Políticas Públicas: Ofrece a gestores ambientales una herramienta de consulta rápida y basada en evidencia para la planificación territorial y la justificación de estrategias de conservación.
  • Conciencia Ciudadana: Sensibiliza a la ciudadanía sobre la vulnerabilidad de su "fábrica de agua", fomentando el apoyo público a la conservación.
